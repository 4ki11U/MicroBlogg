# Flask MicroBlogg
----------


Installing
----------

1. Make ure venv : 


    $ python -m venv .venv

2. Activate ure venv : 


    $ .venv\Scripts\activate

2. Upgrading using `pip`: 


    $ python -m pip install --upgrade pip

3. Install out packages :


    $ pip install -r requirements.txt


Running
----------
Run file run.py : 


    $ python ./run.py
